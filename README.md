AI Multi-Source RAG Chatbot Dashboard is an advanced, unified web application built with Streamlit and LangChain that enables users to query, analyze, and translate data across multiple formats.

The application features a modular dashboard that integrates:

PDF Question-Answering & Translation: Parses documents, generates local vector embeddings (FAISS) to minimize API cost, and translates AI responses into 15+ target languages.
CSV/Excel Data Agent: Automatically previews dataset metrics, renders interactive data visualizations (Altair), and features an AI data agent that writes, corrects, and runs Python analysis queries locally.
Website Reader: Crawls and indexes public URLs to support contextual conversations about webpage content.
